# Harvard Agentic Science -- Shared Skills

This is the shared skills repository for [Harvard Agentic Science](https://github.com/Harvard-Agentic-Science). Skills are reusable agent files (`.md`) that give [Claude Code](https://docs.anthropic.com/en/docs/claude-code) or Codex a specific role and set of instructions. Drop a skill into your project and the agent picks up the persona, workflow, and output format defined in the file.

Browse the `skills/` directory to see what is available. Each skill folder contains a Markdown file with a description, usage guidance, and structured instructions.

## Getting started

1. Find a skill in `skills/` that fits your task.
2. Copy the `.md` file into your project (or reference it directly).
3. Point your agent at the file when you start a session.

## Contributing

Anyone at Harvard can contribute a skill. See [CONTRIBUTING.md](CONTRIBUTING.md) for instructions. The short version: fork the repo, add a folder under `skills/`, and open a pull request.
